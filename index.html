Understanding Standard Costing and Variance Analysis

Variance analysis helps businesses measure the difference between expected and actual performance. This webpage allows users to calculate material variances and labour variances with a downloadable report option for detailed results. The tool helps companies analyse production performance, pinpoint inefficiencies, and identify areas for improvement.

Some Important Variances

Material Price Variance

Formula: (Actual Price - Standard Price) x Actual Quantity

Description: Measures the difference in cost between what was paid for materials and what should have been paid according to the standard.

Material Quantity Variance

Formula: (Actual Quantity - Standard Quantity) x Standard Price

Description: Indicates the difference between the actual quantity of materials used and the standard quantity expected to be used.

Labor Rate Variance

Formula: (Actual Rate - Standard Rate) x Actual Hours

Description: Reflects the difference between the actual labor rate paid and the standard labor rate set by the company.

Labor Efficiency Variance

Formula: (Actual Hours - Standard Hours) x Standard Rate

Description: Shows the difference between the actual hours worked and the standard hours expected for the actual output.

Overhead Variance

Formula: Actual Overhead - Applied Overhead

Description: Measures the difference between the actual overhead costs incurred and the overhead applied based on standard rates.
